## כיצד להצטרף?

אנחנו תמיד שמחים לקבל תורמים חדשים! הנה איך מתחילים:

1. **Fork** — צרו העתק של המאגר לחשבון GitHub שלכם
2. **Clone** — הורידו את הפרויקט למחשב המקומי:
   ```bash
   git clone https://github.com/<your-username>/student-notes.git
   ```
3. **Branch** — צרו ענף חדש לשינוי שלכם:
   ```bash
   git checkout -b feature/my-contribution
   ```
4. **Commit** — בצעו את השינויים ושמרו אותם עם הודעת commit ברורה
5. **Push** — דחפו את הענף למאגר שלכם:
   ```bash
   git push origin feature/my-contribution
   ```
6. **Pull Request** — פתחו PR מהמאגר שלכם למאגר הראשי ותארו את השינוי

> לפני שליחת PR, אנא קראו את קובץ [CONTRIBUTING.md](CONTRIBUTING.md) לפרטים נוספים על סטנדרטים וכללי הפרויקט.
